### Step 1: Install Poetry

**Poetry** is used to manage dependencies and create isolated project environments.

```bash
pip install poetry


# Create a new directory for your project
mkdir iceberg_pyspark_project
cd iceberg_pyspark_project

# Initialize a new Poetry project (follow the prompts, you can accept defaults)
poetry init

# Create a virtual environment within the directory
python -m venv .venv
Platform,Command to Activate
macOS/Linux,source .venv/bin/activate
Windows (Git Bash),source .venv/Scripts/activate
Windows (CMD/PowerShell),.venv\Scripts\activate

poetry add --group=dev pyspark ipykernel

# Install Jupyter (if not already installed globally)
pip install jupyter

# Install the current environment as a Jupyter kernel
python -m ipykernel install --user --name=iceberg_env --display-name "Iceberg PySpark Dev"

# Start the Jupyter Notebook server
jupyter notebook
